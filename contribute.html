Did you compute a new version of NEMO and want to share it?

We appreciate if you can upload your new files together with the code that you use to compute it. This way, we can ease the work for next users and trust your version. Go to the project folder and add your files.

Contribute

If you don't know how to adapt NEMO to your specifications...

... maybe someone else could help you. Detail your specifications as an issue and ask for help.
